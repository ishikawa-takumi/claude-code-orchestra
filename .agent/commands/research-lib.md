## 調査項目

### 必ずWeb検索で確認
- 公式ドキュメント
- GitHub README
- PyPI / npm ページ
- 最新のリリースノート

### 文書化する内容

1. **基本情報**
   - 正式名称，バージョン，ライセンス
   - 公式URL
   - インストールコマンド

2. **コア機能**
   - 主要な機能
   - 基本的な使い方（コード例）

3. **制約・注意点**
   - 既知の制限
   - 他ライブラリとの競合
   - パフォーマンス特性
   - 非同期/同期の注意点

4. **本プロジェクトでの使用パターン**
   - 推奨する使い方
   - 避けるべきパターン

5. **トラブルシューティング**
   - よくあるエラーと解決法
