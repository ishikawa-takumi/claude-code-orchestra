$ARGUMENTS をシンプルにリファクタリングしてください．

## シンプル化の原則

1. **単一責任** - 1関数 = 1つのこと
2. **短い関数** - 20行以下を目指す
3. **浅いネスト** - 早期リターンで深さ2以下
4. **明確な命名** - コメント不要なほど明確に
5. **型ヒント必須** - すべての関数に

## 手順

1. 対象コードで使用しているライブラリを特定
2. `.claude/docs/libraries/` で制約を確認
3. リファクタリング実行
4. テストで動作確認

## 注意

- ライブラリの機能・制約は必ず維持する
- 不明な点はWeb検索で確認
- 動作を変えない（リファクタリングのみ）
